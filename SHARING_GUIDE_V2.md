## Understanding Share Links

### Link Format
- **Notes**: `https://cloudxnote.vercel.app/share/note/{token}`
- **Folders**: `https://cloudxnote.vercel.app/share/folder/{token}`

### Link Security
- âœ… Unique, unpredictable tokens
- âœ… Hard to guess (e.g., `note_1234567890_abc123xyz`)
- âœ… Login required to view
- âœ… Only you can edit the original

### Link Behavior
- **Never expires** - works until you delete the note/folder
- **Always current** - viewers see latest version
- **Requires login** - viewers must have CloudNote account
- **View-only** - viewers cannot edit your content

## Tips & Tricks

### Sharing Tips
1. **Start with private** - you can always make it public later
2. **Copy link immediately** - it's auto-copied when you share
3. **Test the link** - open in incognito to verify it works
4. **Share responsibly** - remember viewers need to login

### Organization Tips
1. **Share whole folders** - easier than individual notes
2. **Use descriptive titles** - help recipients understand content
3. **Keep it updated** - changes sync automatically
4. **Remove when done** - clean up old shares

### Security Tips
1. **Private for sensitive** - use private shares for confidential info
2. **Public for knowledge** - use public for community content
3. **Logout on shared devices** - protect your account
4. **Review periodically** - check what you've shared
