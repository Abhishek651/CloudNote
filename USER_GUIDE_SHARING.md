# User Guide: Sharing Features

## How to Share Notes and Folders

### Sharing a Note

1. **Create or open a note** you want to share
2. Click the **"Share to Global"** button (globe icon)
3. Your note is now visible in the Global feed
4. To get a shareable link:
   - Go to the **Global** page
   - Find your shared note
   - Click the **Share icon** (ðŸ”—)
   - The link is automatically copied to your clipboard
5. Share the link with anyone!

### Sharing a Folder

1. **Navigate to a folder** you want to share
2. Click the **"Share to Global"** button
3. The entire folder structure (including subfolders and notes) is shared
4. To get a shareable link:
   - Go to the **Global** page
   - Find your shared folder
   - Click the **Share icon** (ðŸ”—)
   - The link is automatically copied to your clipboard
5. Share the link with anyone!

## How Share Links Work

### For the Sender
- When you share a note or folder, a unique link is generated
- Example: `https://cloudxnote.vercel.app/share/note/abc123xyz`
- This link never expires (unless you remove the item from global)
- You can copy the link anytime from the Global page

### For the Receiver
1. **Click the share link** you received
2. If not logged in, you'll see a message:
   - "This note/folder is shared with you, but you need to sign in or create an account to view it"
3. **Click "Sign In / Sign Up"**
4. After logging in, you're automatically redirected to the shared content
5. You can now view the note or explore the folder

## Real-Time Updates

### Automatic Syncing
When you update a note or folder that's shared globally:
- **Changes sync automatically** to the global version
- **No manual action needed** - it happens in the background
- **Recipients always see the latest version**

### What Gets Synced
For notes:
- Title changes
- Content updates
- File attachments (PDFs)

For folders:
- Folder name changes
- New notes added
- Notes removed
- Subfolder changes
- Complete structure updates

## Remember Me Feature

### What It Does
- **Checked**: Stay logged in even after closing the browser
- **Unchecked**: Logout when you close the browser

### When to Use
- **Check "Remember Me"** on your personal devices
- **Uncheck "Remember Me"** on shared/public computers

### How to Use
1. On the login page, you'll see a **"Remember me"** checkbox
2. It's **checked by default** for convenience
3. Uncheck it if you're on a public computer
4. Login as usual

## Privacy & Security

### Who Can See Shared Content?
- **Anyone with the link** can view shared content
- **Login required** - viewers must have a CloudNote account
- **View-only** - viewers cannot edit your content
- **You control sharing** - remove from global anytime

### Share Link Security
- Links use **unique, unpredictable tokens**
- Links are **hard to guess** (e.g., `note_1234567890_abc123xyz`)
- Only you can **remove or update** shared content
- **No expiration** - links work until you remove them

## Tips & Best Practices

### Sharing Tips
1. **Review before sharing** - make sure content is ready
2. **Use descriptive titles** - help others find your content
3. **Update regularly** - changes sync automatically
4. **Remove when done** - unshare content you no longer want public

### Organization Tips
1. **Share complete folders** - easier than sharing individual notes
2. **Use tags** - help categorize shared content
3. **Keep it organized** - well-structured folders are easier to navigate

### Security Tips
1. **Don't share sensitive info** - global content is public
2. **Use Remember Me wisely** - only on trusted devices
3. **Logout on shared computers** - protect your account
4. **Review shared items** - periodically check what you've shared

## Troubleshooting

### Share Link Not Working
- **Check if removed**: Verify item is still in Global page
- **Login required**: Recipient must be logged in
- **Copy again**: Try copying the link again

### Updates Not Syncing
- **Wait a moment**: Sync happens within seconds
- **Refresh page**: Reload the Global page
- **Check connection**: Ensure you have internet

### Can't See Shared Content
- **Login required**: Make sure you're logged in
- **Link correct**: Verify the complete link was copied
- **Item removed**: Sender may have removed it from global
