## ğŸ”§ Tips & Tricks

### Keyboard Shortcuts (Desktop)
- **Ctrl/Cmd + B** - Bold
- **Ctrl/Cmd + I** - Italic
- **Ctrl/Cmd + U** - Underline
- **Ctrl/Cmd + Shift + 7** - Ordered list
- **Ctrl/Cmd + Shift + 8** - Bullet list
- **Ctrl/Cmd + K** - Insert link
- **Ctrl/Cmd + Z** - Undo
- **Ctrl/Cmd + Y** - Redo

### Copy & Paste
- Paste from Word, Google Docs, or any website
- Formatting is preserved automatically
- Images can be pasted directly (desktop)

### Formatting Tips
1. **Headings** - Use H1 for main title, H2 for sections, H3 for subsections
2. **Lists** - Press Tab to indent, Shift+Tab to outdent
3. **Links** - Select text first, then click link button
4. **Code** - Use inline code for short snippets, code blocks for longer code
